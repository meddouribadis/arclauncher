Installation

Get Started in Minutes

Three simple steps to transform your Android TV

01

Download the APK

Visit our GitHub releases page and download the latest APK file. We recommend the release version for stability.

02

Enable Unknown Sources

On your Android TV, go to Settings > Security > Unknown Sources. Enable the option to install apps from unknown sources.

03

Install & Launch

Transfer the APK to your Android TV using a USB drive, ADB, or cloud storage. Open it and follow the installation prompts. Set Arc Launcher as your default home app.
