# Arc Launcher Landing Page Specification

## 1. Project Overview
- **Project**: Arc Launcher Landing Page
- **Type**: Static marketing website (HTML/CSS/JS)
- **Core Functionality**: Showcase Arc Launcher features, provide installation instructions, and link to GitHub
- **Target Users**: Android TV users looking for a better launcher

## 2. UI/UX Specification

### Layout Structure
- **Header**: Fixed navigation with logo and GitHub link
- **Hero Section**: Full-width, bold typography, app screenshot showcase
- **Features Section**: Grid of key features with icons
- **Installation Section**: Step-by-step guide with visual indicators
- **Contribute Section**: GitHub contribution call-to-action
- **Footer**: Copyright and links

### Visual Design (Option C: Tech Product)
- **Color Palette**:
  - Primary: `#6366F1` (Indigo 500 - tech/trust)
  - Secondary: `#1E293B` (Slate 800 - dark tech feel)
  - Accent: `#22D3EE` (Cyan 400 - highlights)
  - Background: `#0F172A` (Slate 900 - deep dark)
  - Text: `#F8FAFC` (Slate 50)
  - Muted: `#94A3B8` (Slate 400)
- **Typography**:
  - Headings: 'Space Grotesk', sans-serif (tech feel)
  - Body: 'Inter', sans-serif (readability)
- **Visual Effects**:
  - Subtle glow effects on cards
  - Smooth scroll behavior
  - Hover lift effects on feature cards
  - Gradient overlays on hero

### Components
- **Navbar**: Logo (Arc Launcher), navigation links, GitHub icon button
- **Hero**: Headline, subheadline, CTA button, floating app screenshot
- **Feature Card**: Icon, title, description, hover state
- **Step Card**: Number, title, description for installation
- **GitHub Button**: Prominent CTA with GitHub logo

## 3. Functionality Specification

### Core Features
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling to sections
- Interactive hover states
- Static content (no backend)

### Content
- **Hero**: "Transform Your Android TV Experience"
- **Features**:
  - Customizable Grid Layout
  - Category Organization
  - Sleek Settings Interface
  - Open Source & Free
- **Installation**: Steps to install APK from GitHub releases
- **Contribute**: Link to GitHub repository

## 4. Acceptance Criteria
- Page loads without errors
- All sections visible and styled correctly
- Responsive on mobile and desktop
- GitHub link works
- Installation steps are clear
